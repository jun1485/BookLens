# 📚🎬 북무비 - 나만의 영화/책 리뷰 앱

React Native와 TypeScript로 개발된 영화/책 리뷰 및 추천 앱입니다.

## 주요 기능

- **영화/책 검색**: TMDB API와 네이버 책 API를 활용한 검색 기능
- **리뷰 작성**: 별점과 함께 영화/책에 대한 리뷰 작성
- **컬렉션 관리**: 나만의 컬렉션으로 영화/책 분류
- **추천 시스템**: 비슷한 장르, 평점을 기반으로 한 추천

## 기술 스택

- React Native
- TypeScript
- Expo
- Supabase (실시간 데이터베이스 및 채팅 기능)
